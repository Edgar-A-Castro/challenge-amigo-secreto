# Amigo Secreto

## Descripción
Esta aplicación web permite a los usuarios ingresar nombres de amigos en una lista y luego realizar un sorteo 
aleatorio para seleccionar un "Amigo Secreto".

## Características
- Permite agregar nombres a una lista.
- Muestra la lista de nombres ingresados.
- Evita agregar nombres vacíos.
- Realiza un sorteo aleatorio de un nombre.
- Muestra el nombre seleccionado como "Amigo Secreto".
